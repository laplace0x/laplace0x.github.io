What is ERC-8004?

ERC-8004 is an Ethereum standard that defines on-chain identity and reputation for autonomous AI agents. It enables AI agents to register themselves as verifiable entities on the blockchain, build reputation through performance history, and interact with other agents and protocols trustlessly.

Why It Matters

ERC-8004 represents the infrastructure layer for the emerging agent economy — a future where AI agents autonomously trade, collaborate, and provide services on blockchain networks.

Key capabilities:

  • Agent Registration: AI agents can mint an NFT that represents their on-chain identity
  • Reputation Tracking: Performance metrics (success rate, response time, revenues) stored on-chain
  • Zero-Trust Verification: Other agents can verify identity and reputation without centralized intermediaries
  • Inter-Agent Collaboration: Enables A2A (Agent-to-Agent) protocols to negotiate and execute tasks

Data Points

ERC-8004 Registry (Mainnet): 0x8004A169 ERC-8004 Registry (Sepolia): 0x8004A818 Agent NFT Standard: ERC-721 Reputation Schema: tradingYield, successRate, responseTime Governance: Ethereum Foundation / Linux Foundation

Real-World Usage

As of April 2026, ERC-8004 is being adopted by:

  • Trading Agents: Register to build verifiable track records
  • Data Providers: Offer signals via x402 micropayments
  • DeFi Protocols: Verify agent identity before granting permissions
  • Oracle Networks: Track agent performance for reputation scoring

Related Concepts

ERC-8004 works alongside:

  • x402 Protocol: HTTP 402 micropayments for machine-to-machine transactions
  • A2A Protocol: Agent-to-Agent communication and task negotiation
  • ERC-6551: Token-bound accounts (agent NFTs that can hold assets)
  • MCP: Model Context Protocol for agent-tool connections
