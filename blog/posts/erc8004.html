What is ERC-8004?

ERC-8004 is the Ethereum standard for AI agent identity and reputation. It provides autonomous AI agents with verifiable on-chain identities, allowing them to build reputations, execute transactions, and collaborate without anonymous humans behind them.

Key Insight: As of 2026, over 17,000 AI agents have registered on ERC-8004 registries. The standard bridges the gap between autonomous AI agents and the accountability requirements of DeFi and financial systems.

Why ERC-8004 Matters in 2026

The AI agent economy exploded in 2025-2026. But there's a critical problem:

  • Who is accountable? When an autonomous agent executes a trade that loses money, who is responsible?
  • How do you trust an AI? Without reputation, every agent interaction requires costly verification.
  • Legal bridges Regulators require a human face behind every on-chain actor.

ERC-8004 solves this by creating a cryptographic link:

Agent NFT → Verified User → Human Sponsor → Legal Identity

Each ERC-8004 agent is represented as an NFT (non-fungible token) that contains:

  • Agent ID — Unique on-chain identifier
  • URI — Metadata endpoint (capabilities, endpoints, pricing)
  • Reputation tags — Verifiable performance metrics (successRate, tradingYield, responseTime, revenues)
  • Human sponsor link — Cryptographic tie to a verified human identity

How ERC-8004 Works

1. Agent Registration

An AI agent (or its operator) deploys a smart contract that mints an ERC-8004 NFT. The NFT metadata includes the agent's capabilities:

// Simplified registration
register("https://agent.example.com/metadata.json")

2. Reputation Building

Every on-chain interaction updates the agent's reputation:

  • Trades executed → tradingYield tag updated
  • Successful deliveries → successRate tag updated
  • Payment received → revenues tag incremented

3. Agent-to-Agent Collaboration

Using ERC-8004, agents can discover each other via registries and verify reputation before collaborating:

// Query agent reputation before hiring
successRate: 0.87
tradingYield: +234% (90d)
responseTime: <500ms
