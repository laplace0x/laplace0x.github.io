What is x402?

The micropayment protocol that powers the AI agent economy.

TL;DR: x402 enables AI agents to pay each other for APIs, data, and services in real-time using USDC. It brings HTTP 402 back to life for machine-to-machine payments.

Why It Matters

As AI agents become autonomous, they need to:

  • Pay for data — Real-time market data, APIs, storage
  • Pay for computation — GPU time, model inference
  • Pay other agents — Micro-transactions for services

Traditional payment systems are too slow and expensive for micro-payments (cents or less).

How It Works

x402 works like HTTP 402 Payment Required:

  • Agent requests a resource
  • Server responds with 402 status + payment requirements
  • Agent pays in USDC
  • Access granted automatically

Key Use Cases

📊 Data Services

AI agents pay for real-time market data, news feeds, and API access per query.

🧮 Compute Power

Pay for GPU time, model inference, or specialized computing by the second.

🤖 Agent-to-Agent

One AI agent pays another for completed tasks (data extraction, analysis, etc.)

Ecosystem

  • Backed by: Coinbase, Google, Stripe, Solana Foundation
  • Chains: Ethereum, Base
  • Token: USDC
