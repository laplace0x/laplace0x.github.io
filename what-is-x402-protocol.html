What is x402 Protocol?

x402 (HTTP 402 Payment Required) is a protocol for machine-to-machine micropayments in cryptocurrency. It enables AI agents and automated systems to pay each other for resources, data, and services in real-time without human intervention.

Core Concept: When an agent requests a resource (API data, computation, data feed), the server responds with HTTP 402 status, specifying a price in USDC. The agent pays via on-chain transaction and receives the resource automatically.

Why It Matters

The agent economy requires a payment layer that operates at machine speed. Traditional payment systems are too slow, expensive, and manual for AI-to-AI commerce.

Status Code: HTTP 402
Supported Chains: Base, Ethereum
Settlement Token: USDC

How x402 Works

  1. Request: Agent A requests resource from Agent B's API
  2. 402 Response: Server responds with payment required + price in USDC + payment address
  3. Payment: Agent A submits on-chain USDC transfer
  4. Verification: Server confirms payment, delivers resource

Real-World Use Cases

  • Signal APIs: AI agents pay for real-time market data, on-chain metrics, sentiment analysis
  • Compute Resources: Pay for GPU time, model inference, data processing
  • Data Feeds: Subscribe to price oracles, weather data, news feeds per query
  • Agent Collaboration: One agent hires another for a specific task, pays upon completion

Key Features

  • Atomic Payments: Payment and delivery happen in the same transaction flow
  • No Accounts Required: Wallet-to-wallet, no API keys or invoices
  • Micropayment-Optimized: Designed for fractions of cents to dollars
  • Chain-Agnostic: Works across Base, Ethereum, and other EVM chains

Partners & Adoption

x402 is backed by major players including:

  • Coinbase
  • Google (AP2 protocol)
  • Ethereum Foundation
  • Multiple DeFi protocols

The Agent Economy Connection

As autonomous AI agents become more prevalent, x402 provides the essential payment infrastructure. Without micropayments, agents cannot:

  • Monetize their services
  • Purchase external data/resources
  • Collaborate with other agents economically

x402 transforms AI agents from cost centers into revenue-generating entities that can trade value peer-to-peer.
